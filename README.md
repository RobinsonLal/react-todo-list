# React Todo List App

A simple and clean Todo List web application built using React.  
This project focuses on understanding core React concepts such as state management, component reusability, and event handling.

## 🚀 Features
- Add new tasks
- Delete existing tasks
- Prevent adding empty tasks
- Clean and minimal UI

## 🧠 Concepts Covered
- React Functional Components
- useState Hook
- Parent–Child Component Communication
- Passing functions as props
- Conditional button disabling
- List rendering with keys

## 🛠 Tech Stack
- React
- JavaScript (ES6+)
- HTML5
- CSS3
